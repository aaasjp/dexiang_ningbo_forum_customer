头像组件测试 - 验证 null/无效值处理

测试场景：

1. null 值： （应显示文字头像 "张三"）
2. undefined： （应显示文字头像 "李四"）
3. 空字符串： （应显示文字头像 "王五"）
4. 字符串 "null"： （应显示文字头像 "赵六"）
5. 无效 URL： （应显示文字头像 "孙七"）
6. 有效头像 URL： （应显示图片头像）

修复说明：

  • ✅ 添加了图片加载错误处理（@error 事件）
  • ✅ 过滤了无效的 src 值（null, undefined, "null", "undefined", 空字符串等）
  • ✅ 当图片加载失败时，自动回退到文字头像
  • ✅ 当 src 改变时，重置错误状态
  • ✅ 根据用户名生成彩色背景和文字头像
