# 引导图片功能说明

## 功能概述

在详情页第一次打开自己的问题帖子时，展示引导图片，指导用户如何切换问题状态。

## 实现内容

### 1. 触发条件

- 用户首次打开自己的**提问类型**问题
- 使用 localStorage 记录是否已经展示过引导（`guideShown` 标识）
- 延迟 1 秒后显示引导

### 2. 图片配置

- **图片路径**: `/detail/guide.png`
- **位置**: 全屏居中显示
- **背景**: 半透明黑色遮罩（rgba(0, 0, 0, 0.7)）
- **z-index**: 2000（确保在最上层）

### 3. 图片样式

```css
.guide-overlay {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0, 0, 0, 0.7);
  z-index: 2000;
  display: flex;
  align-items: center;
  justify-content: center;
}

.guide-image {
  width: 100%;
  height: 100%;
  object-fit: contain;  /* 保持图片比例，完整显示 */
  max-width: 100vw;
  max-height: 100vh;
}
```

### 4. 交互行为

- 点击引导图片的任意位置可关闭引导
- 关闭后将记录到 localStorage，下次不再显示
- 引导关闭时调用 `closeGuide()` 方法

### 5. 代码逻辑

```javascript
// 页面加载时检查
onMounted(async () => {
  await loadDepartmentData()
  await loadPostData()
  
  // 只在自己的提问帖子上显示引导
  if (isOwnPost.value && postData.value.category === 'help') {
    const guideShown = localStorage.getItem('guideShown')
    if (!guideShown) {
      setTimeout(() => {
        showGuide.value = true
      }, 1000)
    }
  }
})

// 关闭引导
const closeGuide = () => {
  showGuide.value = false
  localStorage.setItem('guideShown', 'true')
}
```

## 图片要求

引导图片（`/detail/guide.png`）应该包含：
1. 指向"解决状态"按钮的视觉引导
2. 说明文字（如"点击此处可切换问题状态"）
3. 可选的"我知道了"按钮或提示文字

图片会以居中方式全屏显示，保持原始比例不变形。

## 注意事项

1. 图片文件需要放在 `public/detail/` 目录下
2. 图片应该设计为透明背景或与页面背景融合
3. 图片中应该清晰标注出按钮的位置
4. 建议图片尺寸为 750px 宽度（适配移动端设计稿）
5. 用户关闭引导后，除非清除浏览器缓存，否则不会再次显示

## 测试方法

1. 清除 localStorage: `localStorage.removeItem('guideShown')`
2. 刷新页面，打开自己的提问类型问题
3. 等待 1 秒后引导图片应该出现
4. 点击图片任意位置，引导应该关闭
5. 再次刷新页面，引导不应该再出现
