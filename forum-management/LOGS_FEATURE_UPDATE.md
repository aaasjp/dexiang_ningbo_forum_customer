## 🚀 快速开始

### 访问方式
1. 启动项目：`npm run dev`
2. 在左侧菜单点击 **"操作记录"**
3. 点击 **"查看操作记录"** 按钮

### 使用功能
- **搜索**：输入用户工号后点击搜索或按 Enter
- **分页**：使用页码或每页数量下拉框切换
- **清空**：点击搜索框的清除按钮

## 📊 数据展示

表格包含以下字段：
- 序号
- 用户工号
- 用户姓名
- 操作类型（中文显示）
- 操作内容
- IP地址
- 操作时间

## 🔌 API 接口

**接口**: `GET /api/admin/logs/list`

**参数**:
- `user_code`: 用户工号（可选）
- `page`: 页码（默认1）
- `page_size`: 每页大小（默认10）

详见：`docs/API_DOCUMENTATION.md` 第 9.14 节
